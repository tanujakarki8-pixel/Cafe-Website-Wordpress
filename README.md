# The Cozy Spoon Café Website

## 📖 Project Overview
The Cozy Spoon is a WordPress-based café website designed to enhance customer engagement.  
It provides:
- Interactive menu with images and descriptions
- Recipe sharing section
- Online reservation system
- Responsive design for desktop and mobile

## 🎯 Objectives
- User-friendly interface
- Online reservation system
- Menu display and recipe sharing
- Responsive design
- Enhanced customer engagement

## 🛠 Tech Stack
- WordPress + Elementor
- WPForms (reservations)
- Contact Form 7 (feedback)
- Smush (image optimization)
- MySQL (WordPress backend)
- Canva/Photoshop (design assets)

## 📂 Features
- Homepage with café ambiance and featured items
- Menu section with categories (Beverages, Snacks, Desserts)
- Recipe section with step-by-step instructions
- Reservation form with validation (date, time, guests)
- Contact/Feedback form with integrated map

## ✅ Testing
- Input validation for name, email, guests, date/time
- Error messages for invalid inputs
- Success messages for reservations and feedback
- Responsive design tested on multiple devices
